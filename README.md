📺 YouTube Clone | React.js & Tailwind CSS | Video Streaming App
🚀 Welcome to my YouTube Clone project! This is a video streaming web application inspired by YouTube, built using React.js and Tailwind CSS for a modern and responsive UI.

🔹 Features:
✅ Fetches real-time video data from an API
✅ Modern and fully responsive UI with Tailwind CSS
✅ React Router for seamless navigation
✅ Optimized performance with lazy loading and API caching
✅ Dynamic video recommendations

🛠 Tech Stack:
🔹 Frontend: React.js, Tailwind CSS, Redux (for state management)
🔹 Backend: Firebase (for authentication and database)
🔹 Tools: Git, GitHub
